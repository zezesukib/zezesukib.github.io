# zezesukib.github.io
这是一个简单的静态网页：**年末倒计时**，用于显示从当前时间到本地时区的“12 月 31 日 24:00”（等同于下一年 1 月 1 日 00:00）的实时倒计时。

如何预览：

- 直接打开 `index.html`（双击或在浏览器中打开文件）✅
- 使用简单的本地服务器（推荐）：

  ```bash
  # 在项目根目录运行：
  python -m http.server 8000
  # 然后在浏览器打开 http://localhost:8000
  ```

- 将仓库部署到 GitHub Pages：把 `index.html` 推到 `gh-pages` 分支或在仓库设置中启用 Pages（若使用 GitHub Actions，可进一步自动化）。

说明：倒计时使用浏览器本地时间（用户时区），在目标时间到达时会显示“倒计时结束，祝你新年快乐！”。
